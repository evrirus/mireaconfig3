## Формат входного файла
`A,B,C,D`
- A — opcode
- B,C,D — операнды (количество зависит от команды)

| A  | Команда    | Формат   |
| -- | ---------- | -------- |
| 28 | LOAD_CONST | 28,B,C   |
| 24 | LOAD_MEM   | 24,B,C,D |
| 17 | STORE      | 17,B,C   |
| 19 | SQRT       | 19,B,C,D |

## Пример команды
28,32,536 \
24,779,517,817\
17,495,292\
19,831,600,499
